# QAExam

## Setup
 You must install Node.js on your system. 
 You can download Node.js from [here](https://github.com/user/repo/blob/branch/other_file.md)

```
https://github.com/smf1995/QAExam.git
```

```
npm install
```

(Only for IDE)
